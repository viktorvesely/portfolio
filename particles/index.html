Instructions
This site showcases my current/past/future projects in a semi-interactive manner.
Unfortunately the site will not show properly on mobile devices or on devices with weird screen dimensions.
The site is divided into rows each rows corresponds to a project. You can switch between rows with the up/down arrow keys
Using the left/right arrow keys progresses the project's story.
Try pressing the right arrow key
Instructions #2
Almost all projects your are seeing here are realtime simulations, meaning that they can differ each time you switch between projects.
The particles you are seeing in the background can be controlled using the menu in the upper-right corner. The menu can be open by clicking the "Open Controls" button.
Try dragging the parameters and the simulation will automatically apply them. Have fun with it!
Press the right arrow key to hide all the text fields ( brings them back).
